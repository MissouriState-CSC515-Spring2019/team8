Welcome to Team 8's NASA web app

We built this app with Angular, and are using three NASA web apis:

  • The NASA Mars Rover Photos API

    • Collects image data from NASA's Curiosity, Opportunity and Spirit rovers on Mars

  • The NASA Near Earth Orbit Web Service

    • Used for the retrieval of near earth asteroid information

  • The Astronomy Picture of the Day API

    • NASA's most popular API
